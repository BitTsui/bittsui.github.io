部署

手机服务器可以作为暂时的服务器，新建一个生活博客，部署到手机服务器。

github pages可以作为长期服务器，发布技术方面的博客。

手机服务器+内网穿透

这一种方式挺新鲜的，但是也有不可避免的缺点：

  • 手机需要一直开着，不适合做长期服务器，可以作为暂时服务器使用。
  • 内网穿透用的别人提供的免费版，不稳定，容易崩。
