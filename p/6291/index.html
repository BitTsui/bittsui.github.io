无线连接

Scrcpy 使用 adb 与设备通信，并且 adb 支持通过 TCP/IP连接到设备:

  1. 将设备和电脑连接至同一 Wi-Fi。

  2. 打开 设置 → 关于手机 → 状态信息，获取设备的 IP 地址，也可以执行以下的命令：

  3. 启用设备的网络 adb 功能 adb tcpip 5555。（端口号可以自己设置）

  4. 断开设备的 USB 连接。

  5. 连接到您的设备：adb connect DEVICE_IP:5555 (将 DEVICE_IP 替换为设备 IP).

  6. 正常运行 scrcpy。
